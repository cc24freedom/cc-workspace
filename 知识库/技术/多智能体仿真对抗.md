# 多智能体仿真对抗

## 核心能力
在仿真环境中构建多个智能体进行对抗训练，形成博弈均衡策略。

## 技术架构

### 训练框架
- **算法基础**：[[强化学习算法库]]中的MARL、VDN、QMIX、CTDE
- **对抗机制**：红蓝对抗、自博弈、联盟训练
- **评估体系**：ELO评分、胜率统计、战术多样性评估

### 仿真环境
- **地形规模**：10km×10km三维地形（已验证）
- **节点规模**：≥15个异构装备节点
- **平台类型**：≥5种装备平台
- **交互频率**：≥5Hz高频交互

### 编队协同
- **分层机制**：战略层/战术层解耦
- **动态调整**：节点动态增减、异构混编
- **任务类型**：编队机动、协同侦察、协同打击、巡逻警戒（≥4类）

## 已验证指标

| 指标 | 数值 |
|------|------|
| 推理响应时间 | 17.6ms（训练态）/ 41-50ms（对抗仿真） |
| 指令响应延迟 | 平均110ms，最大≤200ms |
| 节点控制规模 | ≥15个异构装备节点 |
| 支持装备类型 | ≥5种平台 |
