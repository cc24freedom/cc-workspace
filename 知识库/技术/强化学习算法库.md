# 强化学习算法库

## 已验证算法

### 单体算法
- **PPO** (Proximal Policy Optimization) - Actor-Critic + GAE
- **DQN** (Deep Q-Network) - Double/Dueling 变体
- **A3C/A2C** - 异步优势演员-评论家

### 多智能体算法
- **MARL** - 多智能体强化学习框架
- **VDN** - Value Decomposition Networks
- **QMIX** - Q值混合网络
- **CTDE** - 中央训练分散执行范式

### 进阶算法
- **HRL** - 分层强化学习
- **PSRO** - 对抗策略（近似纳什均衡）
