# 每日AI & 军工新闻推送工具

## 项目信息
- **类型**: 个人自动化工具
- **状态**: 运行中
- **代码位置**: `e:\CC\cc-tools\tools\news_daily.py`
- **输出目录**: `e:\CC\项目\news\`

## 功能概述
自动抓取、摘要、推送每日AI和军工领域新闻，支持个性化偏好学习。

## 核心功能

### 1. 多源新闻抓取
**AI新闻源**（8个）：
- 量子位、36氪、虎嗅、雷峰网、InfoQ、钛媒体、少数派、智东西

**军工新闻源**（13个）：
- 中文：观察者网、RFI中文、DW中文、FT中文、自由时报
- 英文：The War Zone、Defense News、C4ISRNET、Politico Defense、Defense One、War on Rocks、MWI

**AI工具动态**：Product Hunt、Hacker News、Reddit

**英文精选**：NYT World、Guardian、Al Jazeera

### 2. 智能过滤
- 深度内容过滤（排除快讯、广告软文）
- 关键词白名单过滤
- 用户屏蔽词过滤
- 最小标题长度限制

### 3. AI摘要生成
- 使用 Claude API 批量生成一句话摘要（20-40字）
- 英文标题自动翻译摘要
- 批量处理（每批15条）

### 4. 今日必读推荐
AI从所有文章中选出当天最值得读的1条，附推荐理由。

### 5. 个性化偏好学习
- 用户在日报中标记 ⭐ 喜欢的文章
- 运行 `--learn` 模式自动提取关键词
- 更新 `preferences.json`，影响后续推送排序

## 技术栈
- **语言**: Python 3
- **依赖**: feedparser, beautifulsoup4, httpx
- **AI**: Claude API (Haiku 4.5)
- **数据格式**: Markdown, JSON

## 使用方式

```bash
# 正常运行（今天已存在则跳过）
python news_daily.py

# 强制重新生成今日文件
python news_daily.py --force

# 扫描所有日报中的 ⭐ 标记，更新兴趣偏好
python news_daily.py --learn
```

## 输出文件
- `e:\CC\项目\news\YYYY-MM-DD.md` - 每日新闻日报
- `e:\CC\项目\news\README.md` - 目录索引
- `e:\CC\项目\tools\preferences.json` - 用户兴趣偏好

## 配置文件
- `news_config.json` - 新闻源配置、关键词、API配置
- `preferences.json` - 用户偏好（自动生成和更新）

## 核心算法

### 偏好评分
```python
score = 0
for kw in boosted_keywords:
    if kw in title: score += 1
if source in boosted_sources: score += 2
```

### 多样性截断
每个类别截断到N条，但保证每个来源至少1条，避免单一来源垄断。

## 特色功能
- **自动学习**: 每次运行前自动扫描前一天的 ⭐ 标记（静默模式）
- **来源多样性**: 优先保证每个来源至少1条，再按评分排序
- **英文支持**: 英文标题自动生成中文摘要
- **今日必读**: AI综合评估选出最值得读的1条
