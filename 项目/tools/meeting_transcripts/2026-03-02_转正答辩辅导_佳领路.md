# 转正答辩辅导对话纪要

**日期**：2026-03-02
**文件来源**：佳领路.m4a
**性质**：售前岗位员工转正答辩前辅导对话

---

## 核心问题诊断

| 问题 | 具体表现 |
|------|---------|
| 内容太具体、太流水账 | 只列出"做了什么"，未体现价值贡献 |
| 措辞被动 | 全程"参与"，看不出主导性 |
| 自我介绍脱节 | 与公司战略目标无关联，未回答"你能带来什么" |

---

## 核心建议

### 1. 先讲价值主张，再介绍自己

领导关心的是你能带来什么，不是你是谁。

- ✅ 开篇："我能帮助团队完成今年1000万战略目标"——领导才会往下看
- ❌ 开篇："我是个写方案的"——没戏，不看了

### 2. 自我定位对标公司战略

售前岗位的核心使命是服务公司战略目标，不能只停留在"体系搭建者"。

- 体系搭建 = 打地基（基石，必要但不够）
- 还要回答：**地基打完后，要盖什么楼，怎么配合公司达成1000万**

### 3. 工作描述要有对比，不要罗列

| 错误写法 | 正确写法 |
|---------|---------|
| "12月16日本人去现场出差" | "主导现场汇报，规避了XX风险，加速了结项" |
| "经我手的项目顺利结项" | "经我手的项目交付率100%；相比之前状态，做到了……" |
| 罗列字数（8.65万字、5.51万字） | 体现独特性：之前是什么状态 → 经我介入后是什么状态 |

### 4. 措辞主动化

- 把"参与"统一改为"主导"、"推动"、"把控"
- 谦逊表达：加"在团队输入和支持下，我主导了……"
- 核心动作必须是你，不能全是"参与"

### 5. 推荐答辩结构

```
① 我的角色定位（对标公司战略，不只是岗位描述）
   → 我是售前，我的核心使命是帮助公司达成1000万目标

② 我能带来什么（先抛结论，引发兴趣）
   → 通过我的售前方案配合，团队交付率达100%

③ 我做了什么（有对比的成果，非工作量罗列）
   → 接手前状态 vs 接手后状态，体现我的贡献

④ 我的未来规划（从打地基到盖楼）
   → 地基阶段：体系搭建✓
   → 下一阶段：提升团队整体水平，传递行业经验
   → 终极目标：支撑公司战略目标实现
```

---

## 其他要点

- 自我介绍与工作概述可以合并为一个部分，结构更紧凑
- 三个项目快速结项是能力体现，可以正面表达："高效推进结项，为团队释放了资源"
- 出差、评审等经历要写成**结果导向**，而非行程记录

## 关键金句（导师原话）

> "我如果说我能给你带来一个亿，他就会看你怎么带来这几个亿。"

> "你所有写的都是参与，这种用词就很不恰当。"

> "你要体现你主导的，而不是参与的。"
