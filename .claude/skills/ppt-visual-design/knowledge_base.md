## Python 模板库研究

### 可用模板库对比

| 库名 | 占位符语法 | 数据源 | 特色功能 | 适用场景 |
|------|-----------|--------|---------|---------|
| pptx-template | `{sales.0.june}` | JSON/CSV | 嵌套数据结构 | 数据驱动报告 |
| templatepptx | `$this$` | Python dict | 简单替换 | 快速生成 |
| pptx-renderer | `{{{var}}}` | Python | 备注嵌入代码 | 动态内容 |
| python_pptx_interface | 函数调用 | 编程 | 复杂操作 | 自定义生成 |

### 推荐方案

**当前采用**：直接使用 python-pptx + 自定义函数库
**优势**：完全控制、无依赖、灵活性高
**未来可选**：集成 templatepptx 用于简单场景

## 2026 年 PPT 设计风格体系

### 三大核心风格

#### 1. 极简风格（Minimalist）

**视觉特征**：
- 大量留白（负空间优化）
- 有限配色（2-3种颜色）
- 扁平化设计
- 戏剧性排版

**布局参数**：
- 标题：32-44pt，粗体
- 正文：24-32pt
- 图片：居中，width=5.5-6英寸
- 间距：top间距≥1.5英寸

**配色方案**：
- 商务：深灰 #2C3E50 + 蓝 #3498DB
- 科技：深蓝 #1E3A8A + 橙 #F97316
- 学术：黑 #000000 + 灰 #6B7280

**适用场景**：商务汇报、学术演讲、投资路演

**当前 AI Gamebot PPT 属于此类**

#### 2. 商务风格（Corporate）

**视觉特征**：
- 中性色调
- 数据可视化优先
- 品牌一致性
- 结构化内容

**关键元素**：
- 图表、信息图
- 公司 Logo 水印
- 统一字体系统
- 清晰的视觉层级

**布局参数**：
- 标题：28-36pt
- 正文：18-24pt
- 图表：占页面 60-70%
- 数据标注：14-16pt

**配色方案**：
- 保守：深蓝 + 灰 + 白
- 现代：黑 + 金 + 白
- 科技：深紫 + 青 + 白

#### 3. 创意风格（Creative）

**视觉特征**：
- 大胆配色（霓虹色、渐变）
- 原创插图/涂鸦
- 3D 元素/玻璃态
- 非对称布局

**布局参数**：
- 标题：可变，44-64pt
- 正文：20-28pt
- 图片：非居中，创意裁剪
- 元素重叠、分层

**配色方案**：
- 霓虹：黑底 + 荧光绿/粉/蓝
- 渐变：双色或三色渐变
- 复古：棕褐色调

**适用场景**：创意提案、品牌发布、营销活动

## 2026 年设计趋势

### 技术趋势

1. **AI 原生集成**
   - 液态布局（自适应文本长度）
   - 智能元素重排

2. **多格式流动性**
   - 16:9（标准演示）
   - 9:16（移动端）
   - A4（打印报告）

3. **交互式幻灯片**
   - 可点击菜单
   - 非线性导航
   - 实时投票

### 视觉趋势

1. **暗黑模式 + 高对比度**
   - 减少眼疲劳
   - 突出重点内容
   - 霓虹色强调

2. **超大字体**
   - 标题占 50%屏幕
   - 一页一个大观点
   - 微内容幻灯片

3. **数据叙事**
   - 一图一洞察
   - 标题即结论
   - 注释突出关键点

4. **后极简主义**
   - 在极简基础上增加精致元素
   - 保持清晰度的同时增加视觉趣味

## 评审标准调整历史

### 初始标准（2026-03-06）

**严重问题阈值**：
- 单页正文>100字
- 完全重复：两页核心内容完全相同
- 逻辑断裂：相邻页面主题跳跃

**优化建议阈值**：
- 单页正文60-80字
- 部分重复：不同页面提到相同数据

## 常见问题与解决方案

### 问题1：评审后是否需要用户确认？

**答案**：不需要。评审有完整执行权限，发现问题后立即修改。

### 问题2：评审可以修改哪些内容？

**答案**：所有内容，包括：
- 文案文字
- 图片（重新生成）
- 图片风格（调整 prompt）
- 布局参数
- 配色方案
- 字体大小

### 问题3：评审修改后是否需要再次评审？

**答案**：是的。修改后自动进入下一轮评审，直到通过标准。

### 问题4：如何选择设计风格？

**答案**：根据场景自动判断：
- 技术方案/商务汇报 → 极简风格
- 数据报告/公司汇报 → 商务风格
- 品牌发布/创意提案 → 创意风格

## 最佳实践

1. **评审即执行**：发现问题立即修改，不等待用户指令
2. **迭代直到通过**：结构评审最多2轮，视觉评审至少1轮
3. **记录改进轨迹**：每次迭代记录改进项，便于用户了解变化
4. **图片质量优先**：Gemini 图片不满意时立即重新生成，不将就
5. **风格一致性**：同一 PPT 内保持统一的设计风格
6. **数据可视化**：一图一洞察，标题即结论
7. **字体层级**：标题 32-44pt，副标题 18-24pt，正文 16-18pt
8. **留白充足**：避免信息过载，每页一个核心观点
