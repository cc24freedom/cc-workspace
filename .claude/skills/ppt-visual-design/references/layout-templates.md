# 版式布局模板

## 封面页

**布局结构**：
- 上部：项目名称（大标题，44-54pt）
- 中部：副标题/汇报主题（28-32pt）
- 中下：关键视觉元素（背景图/概念图）
- 底部：汇报人、单位、日期（18-20pt）

**设计要点**：
- 背景图透明度60-80%，确保文字清晰
- 标题居中或左对齐（根据风格）
- 留白充足，不要拥挤

## 目录页

**布局结构**：
- 左侧：章节列表（编号 + 标题）
- 右侧：装饰性图形或小图

**设计要点**：
- 章节编号用强调色
- 行间距1.5-2倍
- 当前章节可用不同颜色标识

## 章节页

**布局结构**：
- 大标题：章节编号 + 章节名称
- 背景：主色色块或渐变
- 可选：章节概述（1-2句话）

**设计要点**：
- 视觉冲击力强，区分章节
- 文字少，重点突出

## 内容页 - 左右分栏

**布局结构**：
- 左侧（40%）：文字内容（要点列表）
- 右侧（60%）：图表/图片

**设计要点**：
- 左侧文字3-5个要点
- 右侧图表占满空间
- 中间留白分隔

## 内容页 - 上下分栏

**布局结构**：
- 上部（30%）：标题 + 简短说明
- 下部（70%）：大图表/架构图

**设计要点**：
- 适合展示复杂架构图
- 图表要清晰，标注明确

## 数据展示页

**布局结构**：
- 核心数据：大字号居中（48-60pt）
- 辅助指标：3-4个小卡片并列
- 数据说明：底部小字

**设计要点**：
- 数字用强调色
- 单位/标签用辅助色
- 卡片用浅色背景区分

## 架构图页

**布局结构**：
- 标题（顶部）
- 架构图（占满中部）
- 图例/说明（底部或侧边）

**设计要点**：
- 图表要大，清晰可读
- 层次用颜色区分
- 箭头/连接线清晰

## 时间规划页

**布局结构**：
- 横向时间轴或甘特图
- 里程碑节点标注
- 当前进度标识

**设计要点**：
- 时间轴用主色
- 里程碑用强调色
- 已完成/进行中/未开始用不同颜色

## 团队介绍页

**布局结构**：
- 网格布局：照片 + 姓名 + 职责
- 或列表布局：左侧照片，右侧信息

**设计要点**：
- 照片统一尺寸
- 核心成员可突出显示
- 团队规模数据可视化
