# 笔记本环境还原指南

## 1. 克隆工作环境

```bash
# 克隆主工作空间（知识库、项目概览、工具脚本）
git clone https://github.com/cc24freedom/cc-workspace.git e:\CC
cd e:\CC

# 克隆 cc-tools 仓库（独立工具仓库）
git clone https://github.com/cc24freedom/cc-tools.git cc-tools
```

## 2. 安装 Obsidian

1. 下载 Obsidian：https://obsidian.md/download
2. 打开 Obsidian，选择 `e:\CC` 作为 vault
3. 知识库笔记会自动加载

## 3. 配置 Claude Code

### 3.1 安装 Claude Code
```bash
# 如果还没安装
npm install -g @anthropic-ai/claude-code
```

### 3.2 恢复 MEMORY.md
MEMORY.md 已在 `cc-tools/memory/MEMORY.md`，Claude Code 会自动加载。

### 3.3 配置 MCP 工具
复制 `.mcp.json` 到用户目录：
```bash
cp e:\CC\.mcp.json C:\Users\[你的用户名]\.mcp.json
```

## 4. 同步大型交付文档（可选）

大型 .docx/.pptx/.pdf 文件不在 git 中，需要通过以下方式同步：
- 网盘（推荐）：坚果云、OneDrive、百度网盘
- U盘/移动硬盘
- 局域网共享

## 5. 日常工作流

### 台式机 → 笔记本
```bash
# 台式机推送更新
cd e:\CC
git add .
git commit -m "更新：[描述]"
git push

# 笔记本拉取更新
cd e:\CC
git pull
```

### 笔记本 → 台式机
```bash
# 笔记本推送更新
cd e:\CC
git add .
git commit -m "更新：[描述]"
git push

# 台式机拉取更新
cd e:\CC
git pull
```

## 6. 注意事项

- **知识库笔记**（.md）会自动同步
- **项目概览**（项目/*/项目概览.md）会自动同步
- **工具脚本**（.py/.json）会自动同步
- **大型交付文档**（.docx/.pptx/.pdf）需要手动同步
- **临时文件**（~$*.docx、*.tmp）已在 .gitignore 中排除
