# 笔记本环境还原指南

## 1. 克隆工作环境

```bash
# 克隆主工作空间（知识库、项目概览、工具脚本）
git clone https://github.com/cc24freedom/cc-workspace.git e:\CC
cd e:\CC

# 克隆 cc-tools 仓库（独立工具仓库）
git clone https://github.com/cc24freedom/cc-tools.git cc-tools
```

## 2. 安装 Obsidian

1. 下载 Obsidian：https://obsidian.md/download
2. 打开 Obsidian，选择 `e:\CC` 作为 vault
3. 知识库笔记会自动加载

## 3. 配置 Claude Code

### 3.1 安装 Python 依赖
```bash
# 核心依赖
pip install python-pptx python-docx openpyxl PyPDF2

# MCP 工具依赖
pip install faster-whisper==1.2.1 av anthropic

# 可选：Playwright 浏览器自动化
pip install playwright
playwright install chromium
```

### 3.2 安装 Claude Code
```bash
npm install -g @anthropic-ai/claude-code
```

### 3.3 恢复 MEMORY.md
MEMORY.md 已在 `cc-tools/memory/MEMORY.md`，Claude Code 会自动加载。

### 3.4 配置 MCP 工具
`.mcp.json` 已在仓库中，包含以下工具：
- **whisper-local**: 本地语音转写（`项目/tools/whisper_mcp_server.py`）
- **gemini-image**: 图片生成/编辑（`项目/tools/gemini_image_server.py`）
- **doc-reader**: Word/Excel/PDF 读取
- **playwright**: 浏览器自动化
- **git**: Git 操作增强
- **memory**: 知识图谱
- **sqlite**: 数据库操作

MCP 工具会自动从 `.mcp.json` 加载，无需额外配置。

## 4. 同步大型交付文档（可选）

大型 .docx/.pptx/.pdf 文件不在 git 中，需要通过以下方式同步：
- 网盘（推荐）：坚果云、OneDrive、百度网盘
- U盘/移动硬盘
- 局域网共享

## 5. 日常工作流

### 台式机 → 笔记本
```bash
# 台式机推送更新
cd e:\CC
git add .
git commit -m "更新：[描述]"
git push

# 笔记本拉取更新
cd e:\CC
git pull
```

### 笔记本 → 台式机
